## 📝 Product: Office Cat (Social Catalyst for Teams)

### Core Problem
**People don't socialize at work anymore.** Remote work killed spontaneous interaction. Even in offices, people sit with headphones, eat alone, avoid eye contact. Team-building feels forced and awkward. The result: silos, burnout, quiet quitting.

### Core Solution
**Office Cat creates natural excuses for people to interact.** It's a shared digital pet that needs the team to survive. Not a productivity tool. Not a surveillance device. A living reason to talk to each other.

### Key Mechanics
1. **Social Needs** — Cat requires regular human interaction. Left alone, it gets lonely and fades.
2. **Walk to Bond** — Someone picks up the cat, goes for a 10-minute walk with a colleague. Both heal the cat and bond.
3. **Noise Sensing** — Loud offices stress the cat. Team quiets down not because of rules, but because the pet is suffering.
4. **Team Mood Mirror** — Cat reflects collective emotional state. Happy cat = connected team. Fading cat = time to check in.
5. **Shared Rituals** — Feeding, cleaning, seasonal events create recurring social moments.
6. **Quiet Time Insights** — Weekly "team pulse" report: walks taken, interactions, mood trends. No surveillance.

### Emotional Loop
Team ignores each other → Cat gets lonely → Someone notices → "Hey, the cat looks sad, let's take it for a walk" → Two colleagues walk and talk → Cat heals → Team bonds → Cat happy → Repeat
