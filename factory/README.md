# Office Cat — Factory Agent System

## Overview

Factory управляет разработкой продукта Office Cat через специализированных агентов. Каждый агент отвечает за свой домен и работает автономно.

## Agent Roles

### 🤖 CEO Agent
**Status:** Active | **Task:** Product strategy & roadmap
- Определяет vision продукта
- Приоритизирует фичи
- Контролирует метрики
- Отчитывается перед командой

### 💻 Engineer Agent  
**Status:** Active | **Task:** Technical development
- Разрабатывает firmware для Atech.dev kit
- Интегрирует сенсоры (noise, touch, motion)
- Пишет API для dashboard
- Тестирует hardware

### 🎨 Designer Agent
**Status:** Active | **Task:** UX/UI & brand
- Дизайн промо-страницы
- Дизайн dashboard
- Анимации и микро-взаимодействия
- Brand guidelines

### 📊 Analyst Agent
**Status:** Active | **Task:** Market research & sales strategy
- Исследует рынок companion robots
- Анализирует конкурентов
- Строит sales funnel
- Pricing strategy

### 📝 Content Agent
**Status:** Active | **Task:** Copywriting & content
- Тексты для landing page
- Pitch deck content
- Blog posts & social media
- Documentation

### 🚀 Growth Agent
**Status:** Active | **Task:** Go-to-market & distribution
- ProductHunt launch strategy
- LinkedIn campaign
- Pilot program (50 offices)
- Partnerships
