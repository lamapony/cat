## Overview

**Product:** Office Cat — Social Catalyst for Teams
**Version:** 0.1 (Hackathon MVP)
**Date:** May 23, 2026
**Status:** Draft

## User Stories

### As an office manager
- I want to see my team's social health score so I can address isolation
- I want weekly reports without surveillance so I respect privacy
- I want a device that doesn't require IT support so deployment is easy

### As a team member
- I want a reason to talk to colleagues so I feel connected
- I want to help the cat without forced activities so interaction is natural
- I want to see our team's progress so I feel part of something

### As an HR director
- I want measurable wellbeing outcomes so I justify the budget
- I want aggregate data only so I comply with privacy laws
- I want a pilot program so I test before scaling

## Technical Specs

### Hardware (Atech.dev Big Kit)

| Component | Spec | Usage |
|-----------|------|-------|
| Microphone | MEMS, 40Hz-15kHz | Noise sensing |
| Speaker | 2W, 8Ω | Sound effects |
| LED Matrix | 8×8 RGB | Cat face animations |
| Touch Sensor | Capacitive | Pet interaction |
| Accelerometer | 3-axis | Motion detection |
| WiFi | 802.11n | Connectivity |
| Battery | 2000mAh | 8h portable |

### Firmware

```
Main Loop (100ms):
  1. Read sensors (noise, touch, motion)
  2. Update emotion state machine
  3. Render LED matrix
  4. Check connectivity
  5. Send telemetry (aggregate only)
```

### API

```
GET /api/cat/status
→ {health, mood, noise_level, last_interaction}

POST /api/cat/walk
→ {walk_id, start_time, participants}

POST /api/cat/pet
→ {interaction_id, timestamp}

GET /api/team/leaderboard
→ [{name, score, walks, interactions}]

GET /api/team/report?week=2026-W21
→ {walks, interactions, mood_trend, insights}
```

## Design

### Cat States

| State | LED Pattern | Sound | Trigger |
|-------|-------------|-------|---------|
| Happy | Gentle pulse, pink | Purr | High interaction |
| Lonely | Slow blink, grey | Sigh | Low interaction |
| Stressed | Rapid flash, red | Whimper | High noise |
| Healing | Glow wave, cyan | Content hum | During walk |
| Sleeping | Dim breathe, blue | Soft snore | After hours |

### Dashboard

- **Hero:** Live cat status, health bars
- **Actions:** Take a Walk, Pet, Coffee Break, Team Pulse
- **Feed:** Real-time team interactions
- **Leaderboard:** Weekly social scores
- **Report:** Weekly wellbeing summary

## Metrics & Analytics

### Product Metrics
- Daily Active Users (DAU)
- Walks per week
- Interactions per day
- Health score distribution
- Mood trend over time

### Business Metrics
- Waitlist signups
- Pilot retention
- NPS score
- Conversion to paid
- Churn rate

### Privacy
- ❌ No individual tracking
- ❌ No location data
- ❌ No content of conversations
- ✅ Aggregate team data only
- ✅ Opt-in participation
- ✅ GDPR compliant
