# Hackathon Vault — Office Cat

## Структура хранилища

```
hackathon-vault/
├── 01-idea/              # Идея, концепция, vision
├── 02-research/          # Исследования рынка, конкурентов, пользователей
├── 03-product/           # PRD, фичи, механики, дизайн
├── 04-business/          # Бизнес-модель, финансы, инвестиции
├── 05-marketing/         # Маркетинг, продажи, контент
├── 06-technical/         # Техническая документация, API, firmware
├── 07-operations/        # Процессы, задачи, дорожная карта
└── 08-archive/           # Архив, черновики, отмененные идеи
```

## Правила наполнения

1. **Всё на английском** — хакатон международный
2. **Markdown формат** — универсальный, версионируемый
3. **Дата в имени файла** — `YYYY-MM-DD-slug.md`
4. **Источники обязательны** — ссылки, цитаты, данные
5. **SEO-агент переводит** — завод (Paperclip) берёт отсюда и адаптирует
