# Промт для создания промостраницы RoboCat (Skotty)

## Контекст

Ты — senior frontend-разработчик и UX-дизайнер. Твоя задача — создать промостраницу (landing page) для продукта RoboCat (бренд: Skotty) на основе материалов хакатона.

## О продукте

**Skotty** — физическое устройство (робот-кот) для офисов, который решает проблему социальной изоляции в современных офисах.

### Проблема
- 68% сотрудников хотят лучших социальных инструментов (Gallup)
- 47% чувствуют изоляцию в open-plan офисах (HBR)
- Команды с сильными связями на 50% продуктивнее (MIT)
- Люди сидят в наушниках, едят в одиночестве, избегают контактов

### Решение
Skotty — "эмоциональное офисное животное", которое:
- Чувствует шум и стресс в офисе
- Реагирует на прикосновения
- Создает повод для общения ("Кот грустный, пойдем его погуляем")
- Отражает коллективное эмоциональное состояние команды
- Работает 100% локально, без облака

### Эмоциональные состояния
| Состояние | Триггер | LED | Звук |
|-----------|---------|-----|------|
| Happy | Тихий офис + прикосновение | Нежный пульс, розовый | Мурчание |
| Lonely | Нет взаимодействия 10+ мин | Медленное мигание, серый | Вздох |
| Stressed | Шум >70dB | Быстрая вспышка, красный | Хныканье |
| Healing | Во время прогулки | Волна свечения, бирюзовый | Довольное бурчание |
| Sleeping | После работы | Тусклое дыхание, синий | Тихий храп |

### Характеристики
- Размер: 15×10×8 см
- Батарея: 2000mAh, 8+ часов
- Дисплей: 1.8" TFT LCD (128×160)
- Микрофон MEMS + 3 зоны касания
- Динамик 8Ω 1W
- 8× RGB LED
- WiFi только для обновлений
- MicroPython, 100% локальная обработка

### Бизнес-модель
- Цена устройства: 500 DKK (~$75)
- SaaS: $29/мес за команду
- Целевой рынок: $12B
- Pre-seed: $250K при valuation $1.5M

### Целевая аудитория
- Office managers, team leads, HR directors
- Компании 10-100 человек
- Open plan, hybrid, creative spaces

## Технические требования

### Структура страницы
1. **Hero** — headline, подзаголовок, CTA, 3D визуализация кота (placeholder для Three.js)
2. **Problem** — карточки проблем (Too Noisy, Too Cold, Too Distant)
3. **Solution** — как Skotty решает (Senses noise, Responds to touch, Voice Gift)
4. **How it works** — 3 шага (Place & Play, Senses & Feels, Connects People)
5. **Emotions** — 4 состояния с иконками и описаниями
6. **Demo** — интерактивная демонстрация или видео placeholder
7. **Tech specs** — таблица характеристик
8. **Pricing** — 500 DKK/unit
9. **Testimonials** — ранние пользователи
10. **FAQ** — 4 вопроса
11. **CTA** — форма заказа
12. **Footer**

### Дизайн-система
- **Цвета**: 
  - Фон: #faf8f5 (warm cream)
  - Акцент: #f4a261 (amber), #e76f51 (coral), #2a9d8f (sage)
  - Текст: #264653 (primary), #6b705c (secondary)
- **Шрифты**: Nunito (display), Nunito Sans (body)
- **Стили**: Мягкие тени, скругленные углы (12-32px), пастельные градиенты
- **Анимации**: Float для кота, bounce для скролла, fade-in на скролл

### Интерактивность
- FAQ аккордеон
- Smooth scroll для anchor links
- Hover эффекты на карточках
- Анимация появления секций при скролле

### 3D Placeholder
- Контейнер 320×320px с dashed border
- Placeholder с 🐱 и текстом "3D Skotty coming soon"
- CSS-кот как fallback
- Готовая функция `window.initSkotty3D(config)` для интеграции Three.js/Spline

### Код
- Чистый HTML + CSS + JS (без фреймворков)
- Single file (или разбитый по секциям)
- Responsive (mobile-first)
- Accessibility: alt-тексты, aria-labels, keyboard navigation

## Тон и стиль текста
- Дружелюбный, теплый, человечный
- Без корпоративного жаргона
- Фокус на эмоциях и социальных связях
- Теглайны: "Your office just got a heart", "Not a toy. A reason to talk."

## Материалы для использования
- Все тексты, метрики, спецификации — из hackathon-vault
- Структура и стили — из robocat-opendesign.html
- 3D placeholder — уже подготовлен

## Выход

Создай полноценный HTML-файл `skotty-landing.html` со всеми секциями, стилями и скриптами. Страница должна быть готова к деплою — открой в браузере и она работает.

- [ ] Тексты соответствуют продукту (не generic)

- [ ] Responsive на mobile
- [ ] Интерактивные элементы работают
- [ ] 3D placeholder виден
